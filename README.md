# zprolog: Writing a prolog interpreter for fun

Welcome to zprolog -- my attempt to write a prolog interpreter for fun. No other goals.

My knowledge on prolog is rusty, as I've only used it in high school and university. For this project I'm consulting University of Toronto's prolog lecture notes.

I'll start with basic features and will add them to this list as I progress:
- [] parsing facts.
- [] querying facts without variables.
- [] querying facts with variables.
